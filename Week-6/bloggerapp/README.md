# Week 6 - Exercise 13: Blogger Application using Lists and Conditional Rendering

## Objective

The objective of this exercise is to understand how React renders lists using the map() function, assigns unique keys to list items, and performs conditional rendering using different techniques.

---

## Learning Objectives

- Understand Conditional Rendering
- Learn React Lists
- Use map() for rendering collections
- Understand the importance of Keys
- Render multiple React Components
- Work with JSX and Functional Components

---

## Technologies Used

- React
- JavaScript (ES6)
- JSX
- Node.js
- npm
- Visual Studio Code

## Expected Output

The application displays:

- Course Details
- Book Details
- Blog Details

Each section is rendered independently using React components and displayed in a three-column layout.

---

## Industry Best Practices

- Always use unique keys while rendering lists.
- Avoid hardcoding large datasets inside components.
- Fetch data from APIs in production applications.
- Create reusable list components whenever possible.
