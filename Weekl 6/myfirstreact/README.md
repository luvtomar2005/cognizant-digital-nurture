# Week 6 - Exercise 1: Create Your First React Application

## Objective

The objective of this exercise is to set up a React development environment, create the first React application using **Create React App**, and display a simple heading in the browser.

---

## Learning Objectives

After completing this exercise, you will be able to:

- Understand the concept of Single Page Applications (SPA).
- Understand the basics of React.
- Set up a React development environment.
- Create a React application using Create React App.
- Understand the basic project structure of a React application.
- Run a React application using the development server.
- Modify and render React components.

---

## Prerequisites

Before starting this exercise, ensure the following are installed:

- Node.js
- npm (comes with Node.js)
- Visual Studio Code

Verify installation using:

```bash
node -v
npm -v
```

## Steps Performed

1. Installed Node.js and npm.
2. Created a new React application using:

```bash
npx create-react-app myfirstreact
```

3. Navigated to the project directory:

```bash
cd myfirstreact
```

4. Opened the project in Visual Studio Code.

5. Modified `src/App.js` to display a welcome message.

6. Started the development server:

```bash
npm start
```

7. Opened the application in the browser:

```
http://localhost:3000
```

---

## Output

The application displays the following heading centered at the top of the webpage:

```
Welcome the first session of React
```
