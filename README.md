# Git Workflow

Development follows a feature branch workflow.

```text
main
│
├── feature/week-1
├── feature/week-2
├── feature/week-3
├── feature/week-4
├── feature/week-5
├── feature/week-6
├── feature/week-7
└── ...
```

For every week:

- Create feature branch
- Complete exercises
- Commit regularly
- Push to GitHub
- Create Pull Request
- Merge into main
- Delete feature branch
