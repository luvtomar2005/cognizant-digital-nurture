# Exercise 1 - PL/SQL Control Structures

## Objective

Practice PL/SQL control structures using:

- FOR LOOP
- IF Statement
- UPDATE Statement
- DBMS_OUTPUT.PUT_LINE

## Scenarios

### Scenario 1

Apply a 1% discount on loan interest rates for customers older than 60 years.

### Scenario 2

Mark customers as VIP when their account balance exceeds 10000.

### Scenario 3

Display reminder messages for customers whose loan is due within the next 30 days.

## How to Execute

1. Open Oracle SQL Developer.
2. Enable DBMS Output.
3. Execute `ControlStructures.sql`.
4. Verify the changes in the `Customers` and `Loans` tables.

## Concepts Covered

- PL/SQL Block
- FOR LOOP
- IF Condition
- UPDATE
- COMMIT
- DBMS_OUTPUT
