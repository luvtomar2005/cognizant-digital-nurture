# Exercise 1 - Logging Error Messages and Warning Levels

## Objective

Demonstrate logging using SLF4J with Logback.

## Technologies

- Java
- SLF4J
- Logback
- Maven

## Log Levels Used

- ERROR
- WARN

## How to Run

```bash
mvn compile
```

Run the `LoggingExample` class from your IDE.

## Expected Output

- Error log message
- Warning log message
