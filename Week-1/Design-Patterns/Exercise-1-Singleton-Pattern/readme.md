# Exercise 1 - Singleton Pattern

## Objective
Implement the Singleton Design Pattern to ensure only one Logger object exists.

## Concepts
- Singleton Pattern
- Private Constructor
- Static Instance
- Static Factory Method

## Output
Only one Logger instance is created.
