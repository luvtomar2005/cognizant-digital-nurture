# Exercise 2 - E-Commerce Platform Search

## Objective

Implement Linear Search and Binary Search to search products in an e-commerce platform.

## Classes

- Product
- SearchAlgorithms
- Main

## Algorithms

1. Linear Search
2. Binary Search

## Time Complexity

Linear Search
- Best Case : O(1)
- Average Case : O(n)
- Worst Case : O(n)

Binary Search
- Best Case : O(1)
- Average Case : O(log n)
- Worst Case : O(log n)

## Conclusion

Binary Search is more efficient for large sorted datasets because it reduces the search space by half at every iteration.
