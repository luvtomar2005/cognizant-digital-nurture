# Exercise 7 - Financial Forecasting

## Objective

Develop a financial forecasting tool using recursion to predict future values based on historical growth rates.

## Concepts Used

* Recursion
* Base Case
* Recursive Calls
* Time Complexity Analysis

## Formula

Future Value = Previous Value × (1 + Growth Rate)

## Example

Current Value = 10000

Growth Rate = 10%

Years = 3

Predicted Future Value = 13310.00

## Output

Predicted Future Value after 3 years = 13310.00
